# SADAF - Voice & Video Translator

SADAF is a premium, real-time voice translator that converts English to Finnish and Finnish to English. It features a live video feed with glassmorphic subtitle overlays.

## 🚀 Features
- **Live Video Feed**: Mirrored camera preview for a seamless experience.
- **Real-time Subtitles**: Synced translation overlays directly on the video.
- **Dual-mode Translation**: 
  - English -> Finnish
  - Finnish -> English
- **Premium Design**: Vibrant yellow and white theme with modern animations.
- **Microphone Integration**: High-accuracy speech recognition via Web Speech API.

## 🛠️ Technology Stack
- **HTML5/CSS3**: Core structure and premium styling.
- **JavaScript**: Real-time logic and API integration.
- **Web Speech API**: Browser-native voice recognition.
- **MyMemory API**: Reliable translation services.

## 📖 How to Use
1. Open `index.html` in a modern browser (Chrome, Edge, etc.).
2. Grant permissions for **Camera** and **Microphone**.
3. Toggle the language direction as needed.
4. Click the **Microphone** button and start speaking.
5. Watch your subtitles appear live on the video!
